# VaultKit

A lighter, mobile-first workspace kit for Obsidian.

VaultKit starts from the make.md codebase and focuses on a cleaner, faster workspace experience for real vaults, especially on mobile. The long-term direction is a more focused workspace layer with redesigned databases, a more useful graph view, and AI features that support note organization without getting in the way.

## Direction

- Mobile-first workspace navigation and editing
- Lighter defaults than make.md
- Database and structured-note workflows that feel easier to maintain
- Graph views built for understanding relationships, not just visual effect
- AI assistance for organization, summaries, links, tags, and properties

## Status

VaultKit is in early fork setup. The first milestone is separating the plugin identity, storage paths, and build pipeline from make.md so it can be tested independently.
